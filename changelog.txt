CASCADE PIPELINE VERSION 1.3
Sept 22, 2008


SUMMARY:

Recommended upgrade.

There are no major changes that require immediate upgrades, but future
scripts (such as real-time delivery analysis pipeline) will require the
changes implemented in this version.

 - Enforced a new naming convention for functions (camel-case)

 - Fixed several bugs in sorttraces
 
 - Recursive directory processing in Gettraces
 
 - Several new filtering criteria (not available in autotrace yet)
 
 - Other minor bug fixes and improvments
 
 - Scripts that are ribosome-sepecific have been seperated into
   the "ribosome" directory.
   
 - Alpha version of Matlab hidden Markov modeling scripts
 
 - Alpha version of real-time delivery analysis pipeline
 
 - Added a script (frethistComparison) to compare the distribution
   of FRET values across multiple conditions.
